## ✨ About the Project

This project started as a command-line C++ expense tracker and later evolved into a browser-based demo using **WebAssembly**.

The goal was to build something practical while practicing:

- C++ fundamentals
- File handling
- Command-line applications
- Clean project structure
- WebAssembly compilation
- GitHub Pages deployment

The app lets users add expenses, view spending summaries, delete entries, and export their data as a CSV file.
